Python turtle can be used to create interesting animations. This project is based on https://github.com/tsangluo14/Firework-basic, which draws a single firework animation on a black background. This project takes one step further and draws 4 fireworks at random positions in random colors simultaneously. Instead of creating a complicated particle system, these firework projects use simple yet creative line animations to achieve a very nice effect.
